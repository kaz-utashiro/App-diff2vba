
# NAME

App::diff2vba - generate VBA patch script from diff output

# SYNOPSIS

    greple -Msubst --diff old.docx new.docx | diff2vba > patch.vba

# DESCRIPTION

**diff2vba** is a command to generate VBA patch script from diff output.
